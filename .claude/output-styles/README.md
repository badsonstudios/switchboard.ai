# Output Styles

Output styles change how Claude Code communicates by adjusting its system prompt.
Project styles live here as `.claude/output-styles/<name>.md`.

- Activate one with `/output-style` (or `/output-style <name>`).
- A sample `concise.md` is included as a starting point.

**Placeholder for now** — we'll flesh these out later. To add a style, copy
`concise.md`, give it a `name`/`description` in the frontmatter, and write the
behavioral instructions in the body.

Reference: Claude Code docs → Output styles.
